可被监听的类, 可以是任何 class 或 object.

若监听这个类, 监听器将会接收所有事件的广播.

所有 Event 都应继承 AbstractEvent 而不要直接实现 Event. 否则将无法广播也无法监听.

广播

广播事件的唯一方式为 broadcast.
