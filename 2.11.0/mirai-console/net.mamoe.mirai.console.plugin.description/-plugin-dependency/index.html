插件的一个依赖的信息.

只有添加依赖后才能使用依赖插件的 API.

被依赖的插件将会在依赖方之前加载. 即若 A 依赖 B, 那么 B 总是会在 A 之前加载.

注意, 若发现循环依赖, 将会拒绝所有参与循环的插件的加载.
