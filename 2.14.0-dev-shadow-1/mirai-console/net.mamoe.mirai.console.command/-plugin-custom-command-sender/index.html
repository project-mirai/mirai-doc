一个来自插件自行实现的 CommandSender.

PluginCustomCommandSender 不一定拥有全部的权限. PluginCustomCommandSender 可以以其他身份执行命令. 默认情况下 PluginCustomCommandSender 以 ConsoleCommandSender 的身份执行命令
