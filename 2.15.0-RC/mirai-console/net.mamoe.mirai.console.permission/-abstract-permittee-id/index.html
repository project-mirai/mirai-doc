AbstractPermitteeId

@Serializable(with = AbstractPermitteeId.AsStringSerializer::class)
sealed class AbstractPermitteeId : PermitteeId

内建的 PermitteeId.

  • 若指令 A 的权限被授予给 AnyMember, 那么一个 ExactMember 可以执行这个指令.
