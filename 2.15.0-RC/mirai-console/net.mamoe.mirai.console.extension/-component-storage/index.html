ComponentStorage

interface ComponentStorage

组件容器, 容纳 Plugin 注册的 Extension.

插件可在 JvmPlugin.onLoad 时提供扩展. 前端可在 MiraiConsoleImplementation.BackendAccess.globalComponentStorage 获取全局组件容器. 目前未允许获取全局组件容器. 如有需求请 提交 issues.

实现细节: 线程安全.
