FriendGroup

@NotStableForInheritance
interface FriendGroup

一个好友分组. 可能同时存在多个相同名称的分组, 但是每个分组的 id 都是唯一的.

要获取一个分组, 可使用 get 根据 ID 获取, 或者使用 asCollection 获取全部分组列表. 也可以通过 Friend.friendGroup 获取一个好友所在的分组.

在每次登录会话中, FriendGroup 的实例是依据 id 唯一的. 存在于同一个分组中的多个好友的 Friend.friendGroup 会返回相同的 FriendGroup 实例. 但当 bot 重新登录后, 实例可能变化.
