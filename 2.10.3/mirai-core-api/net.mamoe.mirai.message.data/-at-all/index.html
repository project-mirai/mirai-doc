Content copied to clipboard

"@全体成员".

非会员每天只能发送 10 次 AtAll. 超出部分会被以普通文字看待.

使用 AtAll

AtAll 是单例, 将 AtAll 实例添加到消息链中即可.

// Kotlin
contact.sendMessage(AtAll + "test")

// Java
contact.sendMessage(MessageUtils.newChain(AtAll.INSTANCE, new PlainText("test")));
