插件管理器.

PluginManager 管理所有 插件加载器, 储存对所有插件的引用 (plugins), 通过 PluginLoader 间接与 插件实例 交互.

插件加载 和 插件启用 等操作都由 PluginLoader 完成. PluginManager 仅作为一个联系所有 插件加载器, 使它们互相合作的桥梁.

若要主动加载一个插件, 请获取能加载该插件的 PluginLoader, 然后使用 PluginLoader.enable

获取插件管理器实例

可通过 MiraiConsole.pluginManager 或 PluginManager.INSTANCE 获取 PluginManager 实例.
