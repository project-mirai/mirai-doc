recallMessage

abstract suspend fun recallMessage(bot: Bot, source: MessageSource)

撤回这条消息. 可撤回自己 2 分钟内发出的消息, 和任意时间的群成员的消息.

Bot 撤回自己的消息不需要权限. Bot 撤回群员的消息需要管理员权限.

Parameters

source

消息源. 可从 MessageReceipt.source 获得, 或从消息事件中的 MessageChain 获得, 或通过 buildMessageSource 构建.

See also

Throws

PermissionDeniedException

当 Bot 无权限操作时抛出
