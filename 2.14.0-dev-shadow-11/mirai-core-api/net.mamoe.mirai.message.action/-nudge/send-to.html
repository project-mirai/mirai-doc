sendTo

suspend fun sendTo(receiver: Contact): Boolean
Content copied to clipboard

发送戳一戳消息到 receiver.

需要使用支持的协议MiraiProtocol.ANDROID_PHONE 或 MiraiProtocol.IPAD. 自 2.10 起才支持使用 IPAD 协议发送.

Return

成功发送时为 true. 若对方禁用 "戳一戳" 功能, 返回 false.

Throws
