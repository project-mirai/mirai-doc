NotYetLoadedPlugin

interface NotYetLoadedPlugin<T : Plugin> : Plugin

代表一个 未完成加载/延迟加载 的插件

此实例仅用于插件加载系统, 当 PluginManager 加载插件时会自动调用 resolve 解析真正的插件实例
