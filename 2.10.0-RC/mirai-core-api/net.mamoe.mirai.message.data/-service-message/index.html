interface ServiceMessage : RichMessage, CodableMessage
Content copied to clipboard

服务消息, 可以是 JSON 消息或 XML 消息.

XML 消息有时候是 SimpleServiceMessage, 有时候是 LightApp. JSON 消息更多情况下通过 LightApp 发送.

建议使用官方客户端发送来确定具体是哪种类型.
