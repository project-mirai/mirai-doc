abstract suspend override fun sendMessage(message: Message): MessageReceipt<NormalMember>
Content copied to clipboard

向群成员发送消息. 若群成员同时是好友, 则会发送好友消息. 否则发送临时会话消息.

单条消息最大可发送 4500 字符或 50 张图片.

Return

消息回执. 可进行撤回 (MessageReceipt.recall)

See also

net.mamoe.mirai.event.events.GroupTempMessagePreSendEvent

net.mamoe.mirai.event.events.GroupTempMessagePostSendEvent
