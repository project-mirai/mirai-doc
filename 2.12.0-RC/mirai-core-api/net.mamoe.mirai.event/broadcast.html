广播一个事件的唯一途径.

当事件被实现为 Kotlin object 时, 同一时刻只能有一个 广播 存在. 较晚执行的 广播 将会挂起协程并等待之前的广播任务结束.

异常处理

作为广播方, 本函数不会抛出监听方产生的异常.

EventChannel.filter 和 Listener.onEvent 时产生的异常只会由监听方处理.
