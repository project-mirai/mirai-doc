获取原始 MiraiConsoleImplementation 实例.

MiraiConsoleImplementation.start 实际上会创建 MiraiConsoleImplementationBridge 并启动该 bridge, 不会直接使用提供的 MiraiConsoleImplementation 实例. MiraiConsoleImplementation.getInstance 获取到的将会是 bridge. 可通过 bridge.origin 获取原始在 start 传递的实例.
