基础数据类型 Value

9 个被认为是 基础类型 的类型:

  • 整数: Byte, Short, Int, Long

  • 浮点: Float, Double

  • Boolean

  • Char, String

注意: 目前这些类型都会被装箱, 由于泛型 T. 在将来可能会有优化处理. Primitive 仅表示一个类型是上面 9 种类型之一.
