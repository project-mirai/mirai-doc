interface ExtensionPoint<T : Extension>

表示一个扩展接入点(扩展类型). 在 Kotlin 由 Extension 的伴生对象实现, 在 Java 可通过静态字段提供.

在注册扩展时需要提供其 ExtensionPoint, ExtensionPoint 也可以用于获取所有注册的扩展 (ComponentStorage.getExtensions).
